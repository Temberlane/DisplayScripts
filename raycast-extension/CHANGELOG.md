# Display Scripts Changelog

## [Configure Displays UI] - {PR_MERGE_DATE}

- Add `Configure Displays` view command — pick your displays from a dropdown instead of copy-pasting UUIDs.
- Identify a display by shaking the cursor on the physical screen (⌘1 / ⌘2).
- Capture MacBook origins by arranging displays in System Settings and pressing ⌘S / ⌘U.
- Read display resolution and refresh rate live from `displayplacer list` so the extension adapts to any display setup.

## [Initial Version] - 2026-06-18

- Add `Toggle Arrangement` command for switching between side-by-side and stacked MacBook + external monitor layouts.
